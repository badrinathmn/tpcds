# tpcds

A port of the TPC-DS data generator to Java.

## Prerequisites

Tpcds requires Java (1.5 or higher; 1.8 preferred), git, maven (3.2.1 or higher).

## Download and build

```bash
$ git clone git://github.com/julianhyde/tpcds.git
$ cd tpcds
$ mvn install
```
